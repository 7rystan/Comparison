The purpose of this graph is to visualize two collumns of data within the same CSV file, as well as load different CSV files with the drop down menu. Each set of data is split up into 10 bars on the graph at a time to make it more visually appealing and readable, using "next 10" and "previous 10" buttons to surf through the data easily (which could easily be edited to load more or less than 10 at a time if the situation called for it with a few simple changes to the javascript). There's also a checkbox implimented to disable or enable the second set of data if the viewer so chooses.

A good example of the intended data of this graph would be to show the unique views and repeat views of different web pages, where the x axis (bottom) is the name of the web pages, and the y axis (left side) is the number of views. One bar would represent the number of total views, and the other bar would represent the number of unique views only (one per IP address). The hover text would just give more detailed information.
